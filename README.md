## <ins> **TinyApp Project** </ins>

##### TinyApp is a Web Application built on Node and Express which takes a regular URL and transforms it into an encoded version, which redirects back to the original URL!

----------
 ### <ins>  **Techologies** </ins>

 > - Web Server: Node.js
> - Middleware: Express
> - Template Engine: EJS


 ---------------
